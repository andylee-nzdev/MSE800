
# Hello Django

A minimal web application built with Django that displays "Hello Django".

## Project Structure

```
Activity1/
├── manage.py
├── mysite/
│   ├── __init__.py
│   ├── settings.py
│   ├── urls.py
│   ├── wsgi.py
│   └── asgi.py
└── hello/
    ├── __init__.py
    ├── apps.py
    ├── views.py
    └── urls.py
```

## How to Run

```bash
pip install django
python manage.py runserver
```

Then open http://localhost:8000 in your browser.

## Usage

Visiting the home page displays "Hello Django".
